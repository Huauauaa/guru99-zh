# 如何通过示例使用 QTP / UFT 中的恢复方案

> 原文： [https://www.guru99.com/uft-qtp-recovery-scenario.html](https://www.guru99.com/uft-qtp-recovery-scenario.html)

使用恢复方案，您可以指示 HP UFT 从运行会话期间在[测试](/software-testing.html)环境中发生的意外事件和错误中恢复。

对于大型测试而言，恢复方案变得至关重要，因为大型测试将无人值守运行并暂停直到执行恢复操作，这会延长测试执行时间。

创建恢复方案涉及 **6 个步骤**

**步骤 1）**在 Micro Focus UFT 中，选择资源> Recovery Scenario Manager。 创建新方案

**步骤 2）**指定触发事件。 触发事件是中断您的运行会话的事件

**步骤 3）**指定“恢复操作”，这是触发发生时您要采取的纠正措施

**步骤 4）**指定恢复后的测试运行选项，这些选项指定如何在 Quick Test Professional 识别出事件并执行所有指定的恢复操作之后继续运行会话。

**步骤 6）**使用文件>设置> Recovery 将恢复方案添加到您的测试中。 测试结果窗口显示了恢复方案的详细信息
